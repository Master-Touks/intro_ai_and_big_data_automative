# Comment mesurer ma COSMIC FUNCTION POINTS et estimer ma TeeShirt size ? 

## Données
- [] Specifications
    - [] Documentations
    - [] Input / Output
    - [] Références normes
    - [] Efforts
    - [] TeeShirt size


## Transformation des données
- ’[] Est-ce que mes données sont "normal" ? 
  - [Temps moyens livraisons / temps de livraison => aberrant / normal] (moyennes)
  - [Par rapport à la moyenne à quelle distance est-ec que je suis ?] (écart type)
- [] Calcul stats pour sélectionner mes données 
### Output : données propre et entrainable pour mon modèle

## Entraine mon modèle
- [] Selectionne des données de tests (20%)
- [] Sépare les données d'entrainements des données de tests (80%)
- [] J'utilise la loi de Pareto
- [] Et je mesure l'erreur (erreur moyenne, erreur au carré)
### Output : quand l'erreur est minime, j'évalue que mon modèle est satisfaisant

##  Livraison 
- [] Est-ce que mon modèle peut estimer la vérité futur ? 
- [] Est-ce que l'erreur qui avait été minimisé est toujours aussi bonne ?


-- Pour un PO, il peut utiliser un LLM pour l'aider à modéliser ses calculs dans son outil.
